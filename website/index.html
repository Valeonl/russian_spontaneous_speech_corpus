Корпус спонтанной русской речи


Проект по созданию и анализу корпуса спонтанной русской речи с семантическими интерпретациями

О проекте

Данный проект представляет собой корпус спонтанной русской речи, собранный в рамках магистерской работы. Корпус содержит аудиозаписи диалогов, их расшифровки и семантические интерпретации.

Об авторе

Локалина Юлия Сергеевна

Магистрант Университета ИТМО

Магистерская программа "Digital Humanities"

Тема работы: "Исследование спонтанной русской речи для подготовки учебно-методических материалов для курсов по русскому языку для иностранных студентов"
