# Корпус русской спонтанной речи

Репозиторий для хранения корпуса данных по русской спонтанной речи

## О проекте

Данный проект представляет собой корпус спонтанной русской речи, собранный в рамках магистерской работы. Корпус содержит аудиозаписи диалогов, их расшифровки и семантические интерпретации.

Веб-интерфейс проекта доступен по адресу: [https://valeonl.github.io/russian_spontaneous_speech_corpus/](https://valeonl.github.io/russian_spontaneous_speech_corpus/)

### Основные возможности веб-интерфейса:
- Просмотр и поиск по корпусу диалогов
- Фильтрация по ролям говорящих и диалогам
- Прослушивание аудиозаписей
- Интерактивная демонстрация примеров диалогов
- Визуализация семантических интерпретаций

## Структура репозитория

```
russian_spontaneous_speech_corpus/
├── website/                    # Веб-интерфейс проекта
│   ├── index.html             # Главная страница
│   ├── styles/                # CSS стили
│   ├── js/                    # JavaScript файлы
│   └── img/                   # Изображения
├── speech_examples/           # Аудиозаписи диалогов
│   ├── conversation_1/        # Файлы первого диалога
│   └── conversation_2/        # Файлы второго диалога
├── transcripts/               # Текстовые расшифровки
│   ├── raw/                   # Исходные расшифровки
│   └── interpreted/           # Расшифровки с интерпретациями
└── data/                      # Дополнительные данные
    ├── metadata.json          # Метаданные корпуса
    └── statistics/            # Статистика и аналитика
```

## Датасет

Корпус включает в себя:
- Аудиозаписи спонтанных диалогов
- Текстовые расшифровки диалогов
- Семантические интерпретации сленговых и неформальных выражений
- Метаданные о говорящих и контексте диалогов

### Примеры диалогов

На веб-сайте проекта доступны демонстрационные примеры диалогов с возможностью:
- Прослушивания аудиозаписей
- Просмотра расшифровок
- Изучения семантических интерпретаций
- Анализа контекста использования различных выражений
